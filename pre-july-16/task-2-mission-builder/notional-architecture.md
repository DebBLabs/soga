# SOGA Notional Architecture — Pre-July-16 Track

Mission Builder and Governance Server are independent, protocol-independent contributions.

Mission Builder structures human intent into mission/context artifacts.

Governance Server evaluates execution-time legitimacy using governance-relevant information from multiple sources.

AAuth, OAuth, GNAP, MCP, Cedar, and enterprise workflows may provide inputs. None is required. None is privileged.

Governance Server produces ALLOW, RESTRICT, or DENY before execution.
